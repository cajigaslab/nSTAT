# nSTAT Example Data

The paper-example dataset is not stored in this Git repository.

Install it with MATLAB:

```matlab
nSTAT_Install('DownloadExampleData',true)
```

Interactive installs can use:

```matlab
nSTAT_Install('DownloadExampleData','prompt')
```

Source:
- Figshare dataset DOI: <https://doi.org/10.6084/m9.figshare.4834640.v3>
- Paper DOI: <https://doi.org/10.1016/j.jneumeth.2012.08.009>
