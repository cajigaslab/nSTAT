# Archived Python Sync Inventory

This directory is historical reference material from the period when MATLAB-to-Python
parity tracking lived in the MATLAB repository.

Active clean-room workflow:

- the standalone Python port lives in [cajigaslab/nSTAT-python](https://github.com/cajigaslab/nSTAT-python)
- MATLAB-driven cross-language verification lives only in:
  - `tests/python_port_fidelity/**`
  - `tools/python/**`

This `porting/python_sync/` subtree is not part of the active MATLAB runtime,
documentation, packaging, or release flow. Keep it only as archived inventory data
unless a future maintenance task explicitly needs to migrate or delete it.
